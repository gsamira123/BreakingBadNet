# BreakingBadNet

BreakingBadNet es una página web estática dedicada a los personajes de la serie *Breaking Bad*. El proyecto ofrece una experiencia visual coherente para explorar perfiles, información y estadísticas de los personajes principales, así como un formulario de registro para fans.

---

## Descripción del proyecto

Este repositorio contiene el código HTML, CSS y recursos gráficos de una mini página web temática de *Breaking Bad*. El objetivo es presentar:

- una página de inicio con tarjetas de personajes
- páginas individuales con biografías y datos clave de cada personaje
- una sección de registro de fans para interactuar con el contenido
- un diseño consistente en toda la web con un tema oscuro y amarillo inspirado en la serie

---

## Tecnologías utilizadas

- ☸️ **HTML5** - Estructura de la página web
- 🎨 **CSS3** - Estilos y diseño visual
- 🖼️ **Imágenes** - Recursos gráficos para personajes y banners
- 🧩 **Bootstrap** - Componentes básicos de navegación y layout
- 📄 **Markdown** - Documentación del repositorio

## Instalación

1. Clona el repositorio:

```bash
git clone <URL_DEL_REPOSITORIO>
cd BreakingBadNet
```

2. Abre `index.html` en tu navegador para ver la página principal.

3. Si deseas trabajar localmente con un servidor, puedes usar una extensión de servidor local o un comando simple de Python:

```bash
python -m http.server 8000
```

Luego abre `http://localhost:8000` en tu navegador.

---

## Estructura del proyecto

El repositorio tiene la siguiente organización principal:

- `index.html` — página de inicio con tarjetas de personajes.
- `css/` — hoja de estilos y recursos CSS.
- `fotos/` — imágenes de personajes y elementos visuales.
- `pages1/` — páginas individuales de personajes y registro de fans.

---

## Uso básico

- Desde la página principal, haz clic en cualquier tarjeta de personaje para ver su perfil.
- Navega entre `Walter White`, `Jesse Pinkman`, `Saul Goodman`, `Hank Schrader` y otros personajes.
- Usa el formulario de registro de fans para completar los campos y simular el registro.
